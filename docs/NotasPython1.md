## Tema: Markdown

Para poner sub-subtítulos: ### texto

### Sintaxis en Markdown

Para poner negritas: **2 asteriscos** o __2 guiones__

Para poner cursiva: *1 asterisco* o _1 guion_

Para salto de línea: agregar dos espacios al final de la línea de texto.

Para listas sin orden: poner un guion antes de los elementos

- elemento
- elemento

Para listas con orden: poner el número con punto (1. , 2.) antes del elemento

1. primer elemento
2. segundo elemento

Para línea de códgio: acento invertido o "ctrl + shift +ñ".

`línea de código`

    bloque de código
    	instr;

Para poner links: 

- entre corchetes el nombre y entre paréntesis la url  [LCG](https://www.lcg.unam.mx/)

- entre menor y mayor que para sólo usar el link <https://www.lcg.unam.mx/>

Para poner imágenes: ! [texto identificación de imagen] (url de imagen)
